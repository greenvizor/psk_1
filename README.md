# psk_1
Функциональность разрабатываемого приложения учета выработки соусов для ООО "ПСК" может быть разделена на несколько ключевых категорий:
–	Учет ингредиентов: Добавление ингредиентов: Внесение информации о каждом ингредиенте, используемом в производстве соусов. Единицы измерения: Указание единиц измерения для каждого ингредиента (граммы, литры, штуки и т.д.). Срок годности: Фиксация срока годности каждого ингредиента для контроля качества. 
–	Учет рецептов: Создание рецепта: Система для разработки и регистрации рецептов для каждого вида соуса. Состав ингредиентов: Спецификация количества и типа каждого ингредиента в составе рецепта. Процесс приготовления: Добавление описания шагов приготовления соуса в соответствии с рецептом. Связь с производством: Указание рецепта, используемого при производстве каждой партии соуса. 
–	Взаимосвязь с приготовлением соусов: Отслеживание расхода ингредиентов: Автоматическое вычитание ингредиентов из инвентаря при каждом приготовлении партии соуса. Контроль качества: Интеграция с системой учета качества для мониторинга качества ингредиентов и конечного продукта.
–	Ввод данных о приготовленных соусах: Название: Задание уникального названия соусу. Количество: Указание количества приготовленного соуса (в весе, объеме или единицах). Номер партии: Фиксация уникального номера партии для отслеживания производственного процесса. ФИО ответственного: Регистрация имени и фамилии сотрудника, ответственного за приготовление соуса. 
–	Ввод данных о фасовке продукта: Тара: Указание используемой тары для упаковки соуса. Количество: Внесение данных о количестве фасованных единиц продукта. ФИО ответственного: Регистрация имени и фамилии сотрудника, ответственного за фасовку. 
–	Ввод данных об оклейке этикетками: Название бренда: Внесение информации о бренде соуса для последующей оклейки. Количество: Фиксация количества банок или упаковок, которые были оклеены этикетками. ФИО ответственного: Регистрация имени и фамилии сотрудника, ответственного за оклейку этикеток.
